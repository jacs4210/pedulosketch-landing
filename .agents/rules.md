# Reglas Core (Core Rules)

## Directrices Globales
- **Stack Tecnológico:** Frontend Puro (Vanilla HTML5, CSS3, JS ES6+).
- **Prohibiciones:** Está estrictamente prohibido introducir frameworks o librerías pesadas como React, Vue, Angular, Tailwind CSS o Bootstrap. Toda la solución debe construirse usando estándares web nativos.
- **Mentalidad UX/UI Premium:** Todas las adiciones de interfaz de usuario deben respirar calidad (microinteracciones, animaciones sutiles, accesibilidad, y semántica estricta). Revisa las directrices de `.geminirules` para mantener el rol consultivo y proactivo de diseño.

## Configuración de Modelos (Model Sizing)
- **Para ejecución o tareas (Task Execution):** Gemini Flash.
- **Para planeación profunda (Planning Mode):** Gemini 3.1 Pro.
