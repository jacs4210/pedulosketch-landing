# PeludoSketch - Landing Page 🎨🐾

Bienvenido al repositorio oficial de **PeludoSketch**, la marca dedicada a inmortalizar a tus mejores amigos a través de ilustraciones personalizadas creadas por **Alejandra Correa**.

Este proyecto es una landing page moderna, centrada en la conversión y optimizada para ofrecer la mejor experiencia visual tanto en dispositivos móviles como en escritorio.

## 🚀 Despliegue en Vivo
Puedes ver la versión funcional del sitio aquí:
👉 **[https://peludosketch.netlify.app/](https://peludosketch.netlify.app/)**

---

## ✨ Características Principales
- **Galería Premium**: Un sistema de *Masonry Grid* (estilo Pinterest) que permite visualizar las ilustraciones de forma dinámica y fluida.
- **Optimización WebP**: Todas las imágenes han sido procesadas al formato WebP para reducir el peso en un ~35%, garantizando una carga ultrarrápida en Netlify.
- **Diseño Responsivo**: Adaptado perfectamente para usuarios que provienen de Instagram (tráfico móvil).
- **Interactividad**: Incluye efectos de *Smooth Scroll*, *Scroll Reveal* para los precios y un *Lightbox* (visualizador) para ver los retratos en detalle.

## 🛠️ Stack Tecnológico
Para mantener el sitio liviano y fácil de mantener, se ha utilizado **Vanilla Web Stack**:
- **HTML5 Semántico**: Para un mejor SEO y accesibilidad.
- **CSS3 Moderno**: Uso de variables de diseño y grid layouts.
- **JavaScript Puro**: Sin librerías externas, asegurando el máximo rendimiento.

---

## 📂 Estructura del Proyecto
- `/assets/gallery/`: Galería de ilustraciones optimizadas en WebP.
- `/assets/brand/`: Identidad de marca (Logo y etiqueta de autor).
- `/css/`: Estilos globales y sistema de diseño.
- `/js/`: Lógica de interactividad y galería.

## 📦 Cómo Correr Localmente
1. Clona este repositorio.
2. Abre `index.html` en tu navegador favorito (o usa un Live Server).
